Clash for Android 使用教程

0. 简介

Clash是一款用 Go语言 开发的支持 Linux/MacOS/Windows/Android 等多平台的代理工具，支持 ss/v2ray（不支持 ssr），支持规则分流（类似于 Surge 的配置）。

由于客户端长期更新中可能出现界面或功能与教程不符，我们建议您把本教程仅作为参考，具体需要根据自己使用过程中熟悉了解。

1. 下载客户端

  下载 Clash 汉化版）      官方github英文版Clash 客户端   
下载安装后从桌面打开
或在目录中打开 Clash for Windows.exe

2. 设置订阅地址

如果你没有不错的选择 我推荐 https://muacloud.github.io/ 这个网站可以购买到比较优质的节点 支持直接在clash上使用

Clsh使用方法 本方法可以订阅ssr/V2ray/tojan


首先点击上方按钮 复制 Clash 订阅

1. 解压下载的软件 打开Clash for Windows.exe

2. 状态栏调出软件首先选常规然后把系统代理关闭

3. 然后 点击配置在输入URL下载配置文件处填入Clash 订阅

4. 点击下载等待下方黑色卡片增加

备注： 如果你下载遇到了网络失败，检查有没有关闭代理，如果还是不行，可以用其他代理软件连上外网。然后下载


3. 选择节点

点击 代理 选项切换到代理节点界面
根据规则链路设置相应的节点

4. 开启代理

点击 常规 选项切换到通用界面
这里注意看一下端口 有没有7890，如果是0，自己手动修改为7890（1-65335任意端口）
打开 系统代理 开关；其他保持默认即可。
一切准备就绪~~~
